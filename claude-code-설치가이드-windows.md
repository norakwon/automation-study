# 윈도우(Windows)에서 클로드 코드(Claude Code) 처음 설치하기
> 개발 지식이 전혀 없는 분을 위한 단계별 가이드

---

## 시작 전에 알아두기

- **터미널(Terminal)** : 명령어를 입력해서 컴퓨터를 조작하는 창입니다. 겁내지 마세요!
- **명령어를 입력할 때는 복사-붙여넣기를 적극 활용하세요.**
- 윈도우에서 터미널에 붙여넣기할 때는 `Ctrl + V` 또는 마우스 오른쪽 클릭 → 붙여넣기를 사용합니다.
- Windows 10 또는 Windows 11이 필요합니다.

---

## 1단계: Windows Terminal 또는 PowerShell 열기

**방법 A: Windows Terminal (권장)**
1. 키보드에서 `Windows키 + X` 를 누릅니다.
2. 메뉴에서 **Windows Terminal** 또는 **Windows PowerShell** 을 클릭합니다.

**방법 B: 검색으로 열기**
1. 키보드에서 `Windows키` 를 누릅니다.
2. `PowerShell` 을 검색합니다.
3. **Windows PowerShell** 을 클릭해서 엽니다.

> 파란색 또는 검은색 창이 뜨면 성공입니다!

---

## 2단계: Node.js 설치

Claude Code를 실행하려면 Node.js가 필요합니다.

### 방법: 공식 홈페이지에서 직접 다운로드

1. 웹브라우저에서 `nodejs.org` 에 접속합니다.
2. **LTS** 버전의 다운로드 버튼을 클릭합니다. (LTS = 안정적인 버전이라는 뜻)
3. 다운로드된 `.msi` 파일을 더블클릭해서 실행합니다.
4. 설치 마법사에서 계속 **Next** → **Next** → **Install** 을 클릭합니다.
5. 설치 완료 후 **Finish** 를 클릭합니다.

### 설치 확인

PowerShell을 **완전히 닫은 후 다시 열고** 아래를 입력합니다.

```
node --version
```

`v22.0.0` 처럼 버전 숫자가 뜨면 성공입니다!

```
npm --version
```

이것도 숫자가 뜨면 됩니다.

> **오류가 난다면**: PowerShell을 완전히 닫고 다시 열어보세요. 그래도 안 되면 컴퓨터를 재시작하세요.

---

## 3단계: Claude Code 설치

PowerShell에 아래 명령어를 입력하고 엔터를 누릅니다.

```
npm install -g @anthropic-ai/claude-code
```

설치 중 뭔가 물어보면 `y` 를 누르고 엔터를 누르세요.

설치가 끝나면 확인합니다.

```
claude --version
```

버전 숫자가 뜨면 성공입니다!

> **"실행 정책" 관련 오류가 난다면** 아래 해결법을 참고하세요.

---

## (오류 발생 시) 실행 정책 오류 해결

윈도우에서는 보안 정책 때문에 외부 프로그램 실행이 막히는 경우가 있습니다.

PowerShell을 **관리자 권한으로** 열고 아래를 입력하세요.

**관리자 권한으로 PowerShell 여는 방법:**
1. `Windows키` 를 누릅니다.
2. `PowerShell` 을 검색합니다.
3. **마우스 오른쪽 클릭** → **관리자 권한으로 실행** 을 클릭합니다.

관리자 PowerShell에 아래를 입력합니다.

```
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

`Y` 또는 `A` 를 누르고 엔터. 그 다음 다시 3단계로 돌아가세요.

---

## 4단계: Anthropic API 키 발급받기

Claude Code를 사용하려면 **API 키**가 필요합니다. API 키는 Claude Code가 AI 서버에 접속할 때 사용하는 일종의 비밀번호입니다.

1. 웹브라우저에서 `console.anthropic.com` 에 접속합니다.
2. 회원가입 또는 로그인을 합니다.
3. 왼쪽 메뉴에서 **API Keys** 를 클릭합니다.
4. **Create Key** 버튼을 클릭합니다.
5. 키 이름을 아무거나 입력하고 (예: `my-claude-code`) **Create Key** 를 누릅니다.
6. 화면에 나타난 키를 **반드시 복사해서 안전한 곳에 저장**하세요. 창을 닫으면 다시 볼 수 없습니다!

> API 키는 `sk-ant-api03-...` 형태로 시작하는 긴 문자열입니다.

> **유료 서비스 안내**: API 사용은 사용량만큼 비용이 발생합니다. 처음 가입 시 무료 크레딧을 제공하는 경우가 있습니다.

---

## 5단계: Claude Code 처음 실행하기

PowerShell에 아래를 입력합니다.

```
claude
```

처음 실행하면 API 키를 입력하라고 합니다. 4단계에서 저장해둔 API 키를 붙여넣기 하고 엔터를 누르세요.

> PowerShell에 붙여넣기: `Ctrl + V` 또는 마우스 오른쪽 클릭

설정이 완료되면 Claude Code가 시작됩니다!

---

## 6단계: 기본 사용법

Claude Code가 실행된 상태에서:

- **질문하기**: 그냥 한국어로 타이핑하면 됩니다. 예: `파이썬으로 hello world 출력하는 코드 만들어줘`
- **종료하기**: `/exit` 를 입력하거나 `Ctrl + C` 를 누릅니다.
- **특정 폴더에서 작업하기**: PowerShell에서 먼저 해당 폴더로 이동한 후 `claude` 를 실행합니다.

```
cd 폴더경로
claude
```

예시 (바탕화면의 내프로젝트 폴더):
```
cd C:\Users\내사용자이름\Desktop\내프로젝트
claude
```

> 폴더 경로를 모를 때: 탐색기에서 폴더를 열고, 주소창을 클릭하면 경로가 나옵니다.

---

## (선택) 더 편한 터미널 사용을 위한 팁

### 폴더에서 바로 터미널 열기
탐색기에서 작업하고 싶은 폴더를 열고, 주소창에 `powershell` 을 입력하고 엔터를 누르면 해당 폴더에서 PowerShell이 열립니다.

### 자주 쓰는 명령어
| 명령어 | 하는 일 |
|--------|---------|
| `cd 폴더이름` | 해당 폴더로 이동 |
| `cd ..` | 상위 폴더로 이동 |
| `dir` | 현재 폴더의 파일 목록 보기 |
| `cls` | 화면 지우기 |

## 문제 해결

### "node는 내부 또는 외부 명령이 아닙니다" 오류
→ PowerShell을 완전히 닫고 다시 열거나, 컴퓨터를 재시작하세요.

### "claude는 내부 또는 외부 명령이 아닙니다" 오류
→ PowerShell을 완전히 닫고 다시 열어보세요. 그래도 안 되면 아래를 입력하세요.
```
npm install -g @anthropic-ai/claude-code
```

### 설치 중 권한 오류
→ PowerShell을 관리자 권한으로 열고 다시 시도하세요. (3단계 위의 관리자 권한 설명 참고)

### API 키 오류
→ API 키를 다시 확인하세요. 키 앞뒤로 공백이 들어가지 않았는지 확인하세요.
